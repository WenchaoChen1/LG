### Q26. [数据来源] 保存 API 提交方式是增量（diff）还是全量？

**背景**：增量提交需要前端维护变更列表（新增/修改/删除分别标记）；全量提交需要后端做 diff 比对。两者对前后端工作量差异显著。

**文档位置**：需求文档 4.7 节

**参考选项**：
- A. 增量提交：前端记录所有变更操作，保存时只提交变更部分（新增列表 + 修改列表 + 删除ID列表）
- B. 全量提交：前端提交当前所有数据的完整快照，后端与数据库对比生成 diff
