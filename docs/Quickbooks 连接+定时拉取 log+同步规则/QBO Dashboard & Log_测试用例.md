## 需求清单

| # | 需求类型 | 需求描述 |
|---|---------|---------|
| R1 | 功能需求 | QBO Dashboard 仅 PGM/PM 角色有权访问 |
| R2 | UI 需求 | QBO Dashboard 列表位于 Portfolio Companies 页面的一个新标签页中 |
| R3 | 功能需求 | Dashboard 展示所有公司连接的列表及其当前状态（绿色/灰色/红色） |
| R4 | 功能需求 | 连接发生问题时可查看详细日志，包括同步错误信息、时间戳及错误载荷（Error Payloads） |
| R5 | 功能需求 | 查看 Dashboard 时能看到连接状态的实时更新，无需手动刷新页面 |
| R6 | 功能需求 | QBO 连接中断时，用户收到电子邮件通知，告知具体问题及解决步骤 |
| R7 | 功能需求 | QBO 连接恢复正常时，用户收到确认邮件，通知连接已重新建立 |
| R8 | 功能需求 | QBO 连接问题解决时，PGM/PM 可在系统日志中查看解决事件记录（含时间戳和具体操作） |
| R9 | 功能需求 | QBO Log 仅开发者角色有权访问 |
| R10 | 数据需求 | QBO 同步时将完整细节存储到日志中，包括时间戳、成功/失败状态及错误负载数据 |
| R11 | 业务规则 | 手动断联后，再次连接不换账号、不换公司，不需要重新配置匹配项 |
| R12 | 业务规则 | 手动断联后，再次连接不换账号、换公司，需要重新配置匹配项 |
| R13 | 业务规则 | 手动断联后，再次连接换账号，需要重新配置匹配项 |
| R14 | 业务规则 | 自动切手动、手动切自动后，重新连接，不需要重新配置匹配项 |
